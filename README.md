# C++ SFML Project Template

This repository contains a C++ project template for SFML 3.0.2, packaged as a single ZIP file.
The ZIP is intended to be used directly as a project template — do not extract it manually.

# How to Use

- Download the ZIP file from this repository

- In your IDE or build system, select “Create project from template” (or equivalent)

- Choose the downloaded ZIP file as the template source

- The IDE will generate a new project based on it automatically

- No manual extraction required.
